# 0.9.0

Version 0.9.0 marks the first beta release and introduces the foundational components of the application, including:

* Command and Control (C2)
  * Builder
  * Client Control
  * Settings Page
  * Action
    * Message Box
    * Shutdown / Restart
    * Run Shell Commands
    * Download and Run Files on Disk
    * Run .NET Files in Memory
* HTTP Server
* Demo Client
